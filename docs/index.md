## 简介

Silver_Bullet 是一个用于自动化处理融易通系统报关流程的 RPA 项目。它能够自动执行登录、查询、下载和上传文件等一系列操作，大大减少了人工重复劳动。

## 常见问题

### 1. 如何安装依赖？

```bash
pip install -r requirements.txt
```

### 2. 如何运行程序？

```bash
python machongRPA.py
```

### 3. 遇到网络超时怎么办？

请检查网络连接是否正常，并确认目标网站是否可以访问。
