# 使用教程

本教程将指导您如何使用 Silver_Bullet 项目。

## 准备工作

在开始之前，请确保已完成安装步骤，并具备以下条件：

1. 已安装所有依赖项
2. 可以正常访问目标网站
3. 拥有正确的账户凭据

## 运行项目

### 1. 直接运行

```bash
python machongRPA.py
```

程序将按顺序执行以下操作：
1. 登录阿里巴巴融易通系统
2. 查询并下载清单文件
3. 登录 EasyChina 系统
4. 上传文件并监控处理状态
5. 下载处理后的文件

### 2. 分步执行

您可以修改 [machongRPA.py](../machongRPA.py) 文件末尾的 `if __name__ == '__main__':` 部分，选择性地执行特定函数。

例如，只执行 EasyChina 相关操作：

```python
if __name__ == '__main__':
    easyChina()
```

### 文件路径

程序会自动创建以下目录用于文件处理：
- `downloaded_manifests/` - 存储从融易通系统下载的原始清单文件
- `completed_manifests/` - 存储已上传到第三方系统的清单文件
- `downloaded_manifests_new/` - 存储从第三方系统下载的处理后文件

## 查看结果

执行完成后，可以在相应目录中查看下载和处理的文件。

## 日志信息

程序使用 loguru 库记录日志信息，运行过程中会在控制台输出相关信息，便于跟踪执行状态。

## 常见问题

### 1. 登录失败

请检查账户信息是否正确，以及网络连接是否正常。

### 2. 文件下载失败

确认目标文件是否存在，以及是否有足够的磁盘空间。

### 3. 程序卡住

可能是网页加载较慢导致，请耐心等待或适当调整代码中的等待时间。
